Crystal Collectors!

You will be given a random number to start this game.
There are 4 crystals below. Each crystal represents a hidden value that you must uncover using your own reasoning.
Clicking on the crystals will add its given value to your score. Each of the values will change with every new game, so no cheating. 🧐
You win by matching your total score to the goal number. If you go over, you will lose the game. Good Luck! ​
